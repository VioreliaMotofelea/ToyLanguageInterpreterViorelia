# 🧩 ToyLanguageInterpreterViorelia

A Java-based interpreter for a custom toy language. Features a JavaFX GUI for visualization of execution.

## ⚙️ Features
- Control flow execution (loops, conditionals)
- File handling and symbol tables
- Execution stack, memory table, and debugging view

## 🛠 Technologies
- **Language**: Java
- **GUI**: JavaFX
- **Concepts**: OOP, Multithreading, Concurrency, Stack & Heap simulation

## 💡 Highlights
- Educational purpose: understand interpreters
- Visual debugging environment
- Well-structured, testable code
